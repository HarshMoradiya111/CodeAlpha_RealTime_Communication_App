# CodeAlpha_RealTime_Communication_App

Task 4 phase 1 scaffold for the CodeAlpha internship real-time communication app.

## Repo layout
- `backend/` - Express + MongoDB signaling/auth API
- `frontend/` - React + Vite client

## GitHub repository name
Use this exact repository name:

`CodeAlpha_RealTime_Communication_App`

## Phase 1 scope
- Project setup
- Backend and frontend skeletons
- Base README and environment files

## Recommended setup
Backend:
1. `cd backend`
2. `npm install`
3. Copy `.env.example` to `.env`
4. Run `npm start`

Frontend:
1. `cd frontend`
2. `npm install`
3. Run `npm run dev`
